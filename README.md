# File-based_Note_Saver
To create a simple application that allows users to save notes permanently and read them later using file handling in Python

This project is a Notes Saver application that allows users to create and view notes. The notes are stored in a text file (notes.txt), ensuring that data is saved even after the program is closed.
The program provides a menu with three options:
• Add Note – Allows the user to enter and save a note
• View Notes – Displays all saved notes
• Exit – Closes the program
When a note is added, it is stored in the file using append mode, which prevents overwriting existing notes. When viewing notes, the program reads all contents from the file and displays them. If the file does not exist, the program handles the error gracefully.
This project demonstrates file handling, loops, functions, and exception handling in Python.
